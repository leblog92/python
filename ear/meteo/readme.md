# 🌤️ Application Météo - Ouest Parisien

Application Python pour obtenir les prévisions météo de l'Ouest Parisien avec synthèse vocale (MP3/TTS).

## ✨ Fonctionnalités

- 📊 Prévisions météo détaillées pour demain
- 🔊 Génération de fichiers MP3 avec la météo
- 🎧 Écoute directe via Text-to-Speech
- 👕 Recommandations vestimentaires personnalisées
- 🔒 Configuration sécurisée avec variables d'environnement
- 📱 Interface en ligne de commande intuitive

## 📋 Prérequis

- Python 3.8 ou supérieur
- Clé API gratuite OpenWeatherMap
